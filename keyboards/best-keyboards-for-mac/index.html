What can a mechanical keyboard not do? With key switches that produce tactile feedback that is unique to the Mechanical keyboard, your typing experience is going to be faster and better than you ever imagined.

Das Model S professional uses the best gold plated Cherry MX switches that allow you to execute every keystroke with a speed that allows for little or no mistakes. It is specifically designed for Mac compatibility and has several Mac-specific function keys, in addition, to the standard command and Alt option key. The keys are laser etched with fonts that are durable and sleek, providing you with a smooth typing experience.

The keys also support a full N-key rollover with the best anti-ghosting technology. Being a wired keyboard, it has two USB 2.0 Hi-speed hub ports and an extra-long two-meter USB cable with a single USB type A connector. This wonderful Mac designed mechanical keyboard features media controls for quick access to media functions. It also features special media keys with blue inscriptions and blue LEDs that are activated when the Num lock, caps lock, and scroll lock is in use. All the admirable features on this keyboard are enclosed in a closet black case that is durable and sturdy.
