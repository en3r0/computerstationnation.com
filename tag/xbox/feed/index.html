]]> Micro USB cable is a type of USB connector, often found on phones and other mobile devices and it is used for connecting those devices to chargers, computers, TVs, cars, and more. Most phones use it for charging the battery, and it also supports one or more types of data connection via the same port.

A single micro-USB port can support many types of connections depending on the phone settings and the device at the other end of the cable. Mass storage mode, MTP, and PTP modes allow a computer to access files and media stored on the phone. In mass storage mode, the phone appears as a disk on the computer when connected via USB, just like a keychain USB flash drive.

Furthermore, files can be moved to and from the phone simply by dragging and dropping files on the computer. Many phones require that this mode be enabled using a settings menu before files can be transferred to or from the computer.

PTP is used for transferring pictures to and from the computer. It is designed to allow multiple devices to have access to the same set of files at the same time, without the risk of the files becoming damaged. MTP only allows simple access to certain types of files e.g audio files.
