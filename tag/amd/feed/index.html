The GA-990FXA-UD3 has the Realtek RTL 8111E chip with a 10/100/1000 network connector. The motherboard is additionally equipped with the ALC889 audio codec that supports Dolby home theatre 7.1 surround sound.

This motherboard also features a Vcore setting starting from -0.600 Volt to +0.600 Volt. This motherboard features a CPU Smart Fan option that gives fan control for a 4-pin CPU fan, a 3-pin CPU fan, and a 4-pin chassis fan.
