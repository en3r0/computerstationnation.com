]]> CPU means Central Processing Unit. It is also called the central processor, main processor, or processor. It is the principal part of a computer and it composes of the main memory, a control unit, an arithmetic- logic unit. It is the heart of the computer and it is linked with input and output devices and auxiliary storage units.

The CPU is contained on an integrated circuit chip called a microprocessor. It regulates the operations of the computer. It receives data from the main memory and interprets them in order to activate other elements of the computer at the appropriate moment to perform their respective operations.

GeForce GTX 1080 is a high-end computer graphics card by NVIDIA, launched in May 2016. Built on the 16 nm process, and it is based on the GP104 graphics processor, in which it GP104-400-A1 variant and the card supports DirectX 12 which ensures that all modern games will run on GeForce GTX 1080.
