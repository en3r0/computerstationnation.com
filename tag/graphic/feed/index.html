To run illustrator, you may want to get a machine with a Multicore Intel processor (with 64-bit support) or AMD Athlon 64 processor, Windows 10 (64-bit) operating system version ranging from V1809, V1903, V1909 or V2004, an 8GB RAM (16GB is recommended), 2 to 4GB HDD (SSD is recommended) and a monitor resolution of 1024Ã—768 (1920Ã—1080 is recommended).

To use the touch workspace in Illustrator, you must have a touch-screen-enabled tablet/monitor running Windows 10 ( e.g. Microsoft Surface Pro). You also need 4GB VRAM for GPU performance and a computer which supports OpenGL version 4.0 or greater.
